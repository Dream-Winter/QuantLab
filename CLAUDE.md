# QuantLab - Hong Kong Stock Quantitative Trading Project

## Project Goals
- Use reinforcement learning (RL) to optimize trading strategies for Hong Kong stocks with small capital
- Current data source: AkShare
- Target: Develop and backtest RL-based trading strategies for Hong Kong stocks
- Initial focus: Tencent Holdings (00700.HK)

## Current Status
- Basic libraries installed: akshare, pandas, matplotlib
- Data fetching module implemented for historical data retrieval
- Next steps: Implement RL environment, agent training, and strategy backtesting

## Data Sources
- Primary: AkShare for Hong Kong stock data
- Future: Consider additional data sources for features and fundamentals

## Development Plan
1. Data acquisition and preprocessing
2. Feature engineering for trading signals
3. RL environment design (state, action, reward)
4. Agent implementation (DQN, PPO, or other suitable algorithms)
5. Backtesting and performance evaluation
6. Strategy optimization and deployment preparation

## Notes
- All data and code should be stored in appropriate directories
- Follow quantitative trading best practices for risk management
- Ensure reproducibility of experiments
